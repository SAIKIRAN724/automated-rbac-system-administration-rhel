# Automated Role-Based Access Control and System Administration in RHEL

This project provides a **menu-driven, automated system administration tool** for Red Hat Enterprise Linux (RHEL).  
It simplifies **user management, group management, sudo privilege control, and LVM operations** using **Bash shell scripting**.
